# Test Plan: Main Information Section

## Overview
This test plan covers the main functionality of the "Main Information" (Основная информация) section of the VM creation form.

## Test Cases

### 1. Verify Name field validation
- **Description**: Submit the form with an empty Name field
- **Expected Result**: Validation error is displayed for the Name field

### 2. Verify Chipset field options
- **Description**: Check that Chipset field's dropdown has all necessary options
- **Expected Result**: "q35" and "i440" options are available for selection

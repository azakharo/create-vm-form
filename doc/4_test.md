# Реализовать первый настоящий тест для формы "Создание ВМ"

Нужно реализовать первый настоящий тест.

Для этого нужно:

* Заполнить обязательное поле "Имя"
* Создать 1 HHD диск
* Нажать кнопку "Создать"
* Сделать mock для функции createVm. Убедиться, что эта функция вызывается 1 раз с правильными значениями (со значениями, введёнными в форму).

## Постановка задачи. Требования

* В файле CreateVm.test.tsx нужно заменить существующий там тест, на тест описанный в этом документе.
* Перед тем, как писать код, нужно задать вопросы, если что-то не понятно.

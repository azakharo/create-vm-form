# Нужно протестировать возможность выбора chipset

Нужно написать новый тест, который проверит, что в форме создания ВМ можно изменить chipset.

Для этого нужно:

* Заполнить обязательное поле "Имя"
* Изменить значение поля Chipset на i440
* Создать 1 HHD диск
* Нажать кнопку "Создать"
* Сделать mock для функции createVm. Убедиться, что эта функция вызывается 1 раз с правильными значениями (со значениями, введёнными в форму).

## Постановка задачи. Требования

* Тест нужно добавить в файл CreateVm.test.tsx.
* Для запуска тестов использовать команду `npm test`.
* Перед тем, как писать код, нужно задать вопросы, если что-то не понятно.

# Реализовать раздел "Диски HDD" формы "Создание ВМ"

Раздел должен выглядеть как обособленная группа полей ввода. Для этого его нужно разместить в отдельной Card.
У раздела должен быть заголовок: "Диски HDD". Рядом с заголовком должна быть кнопка "+" для добавления нового диска HDD.

При нажатии на кнопку "+" в форме появляется новый диск.
Далее пользователь должен указать атрибуты для нового диска.

## Атрибуты диска

| Поле             | Тип                           | Input type          | Обязательное | Начальное значение | Описание                                      |
|:-----------------|:------------------------------|:--------------------|:------------:|:------------------:|:----------------------------------------------|
| Имя              | string                        | Text input          |      Да      | disk-{disk-index}  | Никакой дополнительной валидации не требуется |
| Включен/Выключен | boolean                       | Switch              |      Да      |        true        |                                               |
| Размер           | integer                       | input type="number" |      Да      |       128 ГБ       |                                               |
| Интерфейс        | enum: SCSI, SATA, IDE, VIRTIO | Select              |      Да      |        SCSI        |                                               |

* Если в форме выбран chipset="q35", то для диска нельзя выбрать интерфейс IDE (это значение не должно присутствовать в соответствующем Select).

## Дополнительные требования по отображению диска

* Для диска должен быть показан его порядковый номер (#N).
* Рядом с номером (в той же строке) должна быть кнопка удаления.

## Постановка задачи. Требования

* Нужно создать interface HddDisk для хранения атрибутов диска. Положить его в types.ts.
* Раздел нужно разместить в отдельной папке HddDisks в одной папке с формой. Назвать компонент HddDisks.
* Для ввода атрибутов диска нужно реализовать отдельный компонент HddDisk. Его нужно разместить в отдельной файле в одной папке с HddDisks.
* Раздел HddDisks должен поддерживать работу с динамическим списком дисков. Т.е. в список можно добавлять диски, менять их атрибуты или удалять диски.
* Нужно расширить интерфейс CreateVmFormValues - добавить туда переменную состояния для хранения списка дисков.
* Для новый переменной состояния задать начальное значение (пустой список).
* При добавлении диска нужно реализовать валидацию для полей ввода в соответствии с таблицей выше.
* Для списка дисков нужно реализовать дополнительное правило валидации - пользователю нужно задать минимум 1 HDD диск. Без этого форма не должна делать submit.

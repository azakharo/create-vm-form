# Нужно протестировать раздел "Основная информация"

Нужно написать набор тестов для указанного раздела.
При написании тестов нужно базироваться на плане реализации этого раздела, который находится в файле "plans\main-info-implementation.md".

## Постановка задачи. Требования

* Тесты нужно добавить в файл CreateVm.test.tsx.
* Для запуска тестов использовать команду `npm test`.
* Перед тем, как писать код, нужно задать вопросы, если что-то не понятно.
* Нужно реализовывать тесты по одному. После добавления нового теста, нужно запустить тестирование и убедиться, что все тесты проходят, что в консоли нет warning или ошибок.  После этого можно приступать к реализации следующего теста.
* Добавление каждого теста нужно согласовать со мной: стоит его реализовывать или нет. Не надо начинать реализацию до согласования.

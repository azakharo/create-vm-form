# Реализовать раздел "Основная информация" формы "Создание ВМ"

Раздел должен выглядеть как обособленная группа полей ввода.
У раздела должен быть заголовок: "Основная информация".
Нужно добавить подзаголовок/поясняющий текст "Заполните обязательные поля для создания виртуальной машины".

## Поля

| Поле           | Тип           | Обязательное | Начальное значение | Описание                                       |
|:---------------|:--------------|:------------:|:------------------:|:-----------------------------------------------|
| Имя            | Text input    |      Да      |   пустая строка    | Имя виртуальной машины                         |
| Описание       | Text area     |     Нет      |   пустая строка    | Дополнительная информация о создаваемой машине |
| Количество CPU | Integer input |      Да      |         1          |                                                |
| Объём RAM      | Integer input |      Да      |         4          |                                                |

Поле "Описание" размер textarea - минимум 2 строки. Размер textarea должен увеличиваться, если вводится всё больше текста.

## Постановка задачи. Требования

* Раздел нужно разместить в отдельной папке MainInfo в одной папке с формой. Назвать компонент MainInfo.
* Нужно расширить интерфейс CreateVmFormValues - добавить туда переменные состояния для новых элементов ввода.
* Для новых переменных состояния задать начальные значения (указаны в таблице выше).
* Нужно реализовать валидацию для полей ввода в соответствии с таблицей.
* Перед тем, как писать код, нужно задать вопросы, если что-то не понятно. Затем нужно разработать план реализации. Не начинать кодирования без согласования плана со мной.

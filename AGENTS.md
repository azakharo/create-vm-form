# AGENTS.md

## Описание проекта

Проект - это прототип формы создания виртуальной машины. С технической точки зрения данное приложение - это SPA на React.js.

## Технологический стек

* [React](https://react.dev/) (v19)
* [TypeScript](https://www.typescriptlang.org/)
* [Vite](https://vitejs.dev/)
* **UI Library:** [Ant Design v6](https://ant.design/) — используется как основная библиотека компонентов

## Основные npm команды

* Установка зависимостей: `npm install`
* Разработка: `npm run dev`
* Проверка кода на наличие ошибок Typescript: `npm run ts`
* Линтинг и автоформатирование кода: `npm run lint`

## Правила по работе с кодом

* После изменения файлов исходного кода нужно проверить, нет ли ошибок Typescript в изменённых файлах. Для этого нужно вызвать команду `npm run ts`. Найденные ошибки и warnings необходимо исправить.
* После изменения файла исходного кода нужно вызвать `npm run lint` для изменённых файлов. Это приведёт к автоисправлению кода и автоматическому форматированию кода. Проблемы, которые нашёл eslint и не смог сам устранить, необходимо исправить.
* Для верстки предпочтительнее использовать Flex компонент из antd вместо Space, кроме случаев когда необходимо использовать Space.Compact
